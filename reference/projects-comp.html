도봉라이프 (Dobong Life)

AWS 인프라 구축 및 운영 · (주)유머스트알엔디

서울시 도봉구 지역상권 활성화 앱의 클라우드 인프라를 처음부터 설계하고 구축했습니다. 초기 단일 EC2 구조에서 역할 기반 다중 서버 아키텍처로 전환하고, Terraform을 통한 인프라 코드화를 완료했습니다.

GitHub Actions + OIDC 인증으로 SSH 없는 보안 배포 파이프라인을 구축하고, CloudWatch로 실시간 로그 수집 환경을 마련했습니다. 현재 플레이스토어와 앱스토어에 출시되어 서비스 운영 중입니다.

Featured Project

판옵티콘 (Panopticon)

장애 감지 및 관제 시스템 · UBICOMP LAB

연구실 AI 훈련 데이터 수집 파이프라인의 관제 시스템을 설계하고 개발했습니다. Next.js와 Spring Boot 기반 풀스택으로 구현하고, self-hosted runner를 활용한 On-prem CI/CD를 구축했습니다.
